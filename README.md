# Realistic Shadow Generator

This project implements a realistic shadow synthesis pipeline using HTML Canvas and TypeScript.

## Features
- Directional lighting (angle + elevation)
- Contact shadow (dark, sharp near feet)
- Soft shadow falloff
- Exact silhouette-based shadow
- Optional depth map warping

## Outputs
- composite.png
- shadow_only.png
- mask_debug.png

## Tech
- TypeScript
- Canvas API
- No frameworks

## Run
Open `index.html` in a browser.
